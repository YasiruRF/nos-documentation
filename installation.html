Installation & Setup

Prerequisites

NOS generates Python-based ROS2 nodes and launch files. To use it, you must have a supported version of ROS2 installed on your machine.

  • Operating System: Linux (Ubuntu 22.04 LTS or 24.04 LTS recommended).
  • Python: Version 3.8 or higher.
  • ROS2: Supported distributions: Jazzy Jalisco, Iron Irwini, or Humble Hawksbill.
💡 ROS2 Environment

Before compiling or executing any generated packages, make sure you have sourced your ROS2 installation:
source /opt/ros/humble/setup.bash (replace with your installed distribution).

Installing the NOS Compiler CLI

The NOS compiler CLI can be installed directly from PyPI:

pip install --user nos

Alternatively, you can install the latest developer version directly from the GitHub repository:

pip install git+https://github.com/YasiruRF/nos.git

Environment Variables & PATH Configuration

Python registers console scripts in your user home bin directory (~/.local/bin). Ensure this path is loaded in your shell environment: 

# Append local bin to PATH if it's missing
export PATH="$HOME/.local/bin:$PATH"

To persist this setting, add it to your ~/.bashrc or ~/.profile file.

Verifying the Installation

Ensure that the nos command is recognized and active:

nos --version
# Output: nos 0.1.2
